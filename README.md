Installation

Flash via custom recovery (AnyKernel based).

1. Boot into recovery
2. Flash **EXODUS:Void Kernel ZIP**
3. Wipe cache / Dalvik if u want
4. Reboot

> Backup your boot image.  
> You are responsible for your actions.



Warnings

- Do NOT flash over another kernel without understanding the consequences
- Do NOT expect hand-holding
- Do NOT report issues without logs
- Blind flashing leads to bootloops
